# Сборка APK только с телефона (без ПК)

1. Установи приложение **GitHub** (или **Working Copy**/**Termux**) на телефон.
2. Создай новый пустой репозиторий на github.com (через браузер телефона).
3. Загрузи в него содержимое этой папки (`VoiceChangerApp/*`), включая
   скрытую папку `.github/workflows/build.yml` — она уже готова.
4. Открой репозиторий → вкладка **Actions** → workflow "Build APK" должен
   запуститься автоматически после push (или нажми "Run workflow" вручную).
5. Через 5-10 минут в разделе **Actions → (последний run) → Artifacts**
   появится `voicechanger-debug-apk` — скачай его прямо в браузере телефона.
6. Разархивируй, открой `.apk` файл → разреши "Install unknown apps" для
   Chrome/Files, если попросит → установи.
7. Дальше можно редактировать код прямо в GitHub (веб-редактор или
   мобильное приложение) — каждый push будет автоматически пересобирать APK.
